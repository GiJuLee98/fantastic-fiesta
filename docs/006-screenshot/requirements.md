# 006-screenshot: requirements

1. 개요 [1]
    - 이 프로젝트는 AI Agent가 사용할 수 있는 스크린샷 캡처 및 저장용 PowerShell 스크립트를 작성하는 것을 목표로 한다. [1-1]

2. 입력 파라미터 요구사항 [2]
    - 스크립트는 저장할 디렉토리 경로(Directory Path)를 파라미터로 입력받을 수 있어야 한다. [2-1]
    - 스크립트는 저장할 파일 이름(File Name)을 파라미터로 입력받을 수 있어야 한다. [2-2]

3. 스크린샷 캡처 및 저장 기능 요구사항 [3]
    - 다중 모니터 환경을 지원해야 한다. [3-1]
    - 각 모니터 화면을 개별적으로 캡처하여 PNG 형식 파일로 저장해야 한다. [3-2]
    - 파일 저장 시 이름 포맷은 `파일이름_인덱스.png` 형태로 저장되어야 하며, 인덱스는 0부터 시작한다. [3-3]

4. 출력 및 피드백 요구사항 [4]
    - 스크립트 실행 완료 후 Agent가 인식할 수 있는 정해진 텍스트 포맷으로 결과를 출력해야 한다. [4-1]
    - 출력 포맷은 각 저장된 파일의 경로 및 파일 이름과 해당 파일이 매핑되는 모니터 정보(이름 등)를 포함해야 한다. [4-2]
    - 예시 포맷: `지정경로/지정이름_0.png : AAA모니터` 형태로 라인 단위 출력을 제공해야 한다. [4-3]
    - 스크립트의 전체 실행 성공 여부(또는 실패 시 에러 메시지)가 출력에 명확하게 표기되어야 한다. [4-4]
