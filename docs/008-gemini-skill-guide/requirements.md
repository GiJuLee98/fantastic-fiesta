# 008-gemini-skill-guide: requirements

1. gemini agent용 skill 추가 안내서 작성 요구사항 [1]
    - 안내서 문서의 목적 및 대상을 명시하며, agent skills 개념을 처음 접하는 사람도 쉽게 이해할 수 있는 수준으로 작성한다 [1-1]
    - gemini agent용 skill의 개념 및 필요성을 설명하고, skill 추가 시 생성/수정해야 하는 문서 및 폴더 구조에 대해 명확하게 정의한다 [1-2]
    - 새로운 skill을 생성하고 프로젝트에 추가하는 세부 절차를 상세히 정의한다 [1-3]
    - skill 추가 예시 및 작성된 skill의 검증 방법을 기술한다 [1-4]
